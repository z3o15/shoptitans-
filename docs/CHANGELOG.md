# 更新日志

## [2025-11-22] v1.1.0

### 🎯 新增功能
- **图像注释功能**: 新增在原始游戏截图上添加圆形标记的功能
  - 支持在原图上精确标记匹配的装备位置
  - 可选显示相似度百分比文字
  - 支持自定义圆形颜色、大小和字体
  - 自动生成详细的JSON报告
  - 支持批量处理多个截图

### ⚙️ 配置增强
- **注释配置**: 新增注释相关的配置选项
  - `enable_annotation`: 启用/禁用注释功能
  - `circle_color`: 圆形标记颜色
  - `circle_width`: 圆形边框宽度
  - `font_size`: 注释文字大小
  - `show_similarity_text`: 是否显示相似度文字
  - `auto_generate_annotation`: 是否自动生成注释
  - `annotation_output_dir`: 注释输出目录

### 🖥️ 用户界面
- **新菜单选项**: 在主菜单中添加"11. 生成带圆形标记的原图注释"选项
- **交互式配置**: 支持通过界面选择截图、基准装备和匹配阈值

### 📝 文档
- **使用指南**: 新增 `docs/ANNOTATION_USAGE.md` 详细说明图像注释功能的使用方法
- **API文档**: 更新 README.md 包含图像注释功能的使用示例

### 🧪 测试
- **测试套件**: 新增 `tests/test_image_annotator.py` 测试图像注释功能
  - 图像注释器功能测试
  - 注释配置功能测试
  - 主工作流程集成测试

### 🔧 技术改进
- **代码模块化**: 新增 `src/image_annotator.py` 模块，提供图像注释功能
- **配置管理**: 扩展 `src/config_manager.py` 支持注释配置管理
- **错误处理**: 增强错误处理和用户反馈

## [2025-11-22] v1.0.0

### 🎯 初始版本发布
- **双重算法识别**: 传统dHash算法和高级模板匹配算法
- **智能切割**: 固定坐标切割和轮廓检测切割
- **批量处理**: 支持同时处理多个装备图像
- **配置管理**: 通过config.json管理所有参数
- **三步工作流程**: 获取截图 → 分割图片 → 装备匹配
- **圆形标记**: 在切割后的装备图片上添加圆形标记
